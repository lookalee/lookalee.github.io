유용한 터미널 커맨드 (리눅스):

cd : 디렉토리 변경
cd../../users/download : 디렉토리 내비게이션
cd .. : 디렉토리 한단계 위로
cd~ : 홈 디렉토리로 변경
ls : 모든 폴더 리스트
ls-l : 모든 폴더 리스트 (디테일 함께)
ls-a : 모든 폴더 리스트 (숨겨진 폴더도 보여줌)
Mkd folderName : 새로운 디렉토리 생성
man ls : list 매뉴얼
